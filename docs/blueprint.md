# **App Name**: Savita's Codefolio

## Core Features:

- Professional Introduction: Displays an engaging 'About Me' section with a photo, personalized greeting, and a concise professional summary parsed from the resume, highlighting experience in web development, responsive design, and academic projects, problem-solving, documentation, teamwork, and critical thinking.
- Dynamic Project Portfolio: Automatically lists projects from GitHub, prioritizing those with Vercel deployment links, including titles, descriptions, technologies used, and direct access links. This will include projects like 'Food Munch', 'Wikipedia Search Application', and 'Vehicle-to-Vehicle Communication Using LoRa'.
- Skills & Technologies Showcase: Categorized display of technical proficiencies including Programming (Python, JavaScript), IT Constructs (DBMS, OS, Computer Networks), Development Tools (MS Visual Studio, Google Colab), Databases (MongoDB, SQL), Other (GitHub, HTML5, CSS), and Soft Skills (Problem Solving, Documentation, Teamwork, etc.) to highlight expertise.
- Work Experience & Education Timeline: Presents a clear, chronological overview of professional work history, such as 'Teaching Assistant' at NxtWave Technologies, and academic background from AMC Engineering College, EXCELLENT PU College, and St. Joseph English Medium, complete with roles, responsibilities, achievements (e.g., Quantum Hacks 2025 participation, coding problems solved), and certifications (Responsive Website, Static Website).
- AI-Powered Project Inquiry Tool: An interactive tool allowing visitors to ask questions about specific skills or technologies and instantly see relevant projects from the portfolio where those skills were applied, utilizing generative AI for reasoning.
- Centralized Contact Hub: Provides easy access to contact information and professional social media profiles with direct links for email (teggisavi@gmail.com), LinkedIn (linkedin.com/in/savita-teggi-a15a90307), GitHub (github.com/savitateggi), LeetCode (leetcode.com/u/Savitateggi021), and Instagram (instagram.com/_savii_21/).

## Style Guidelines:

- Color scheme: Dark mode, evoking a professional and modern tech aesthetic.
- Primary color: A vibrant deep violet (#6033CC), conveying creativity and technical expertise against a dark backdrop.
- Background color: A subtle dark charcoal with a hint of violet (#1C1A1F), providing a sophisticated and focused canvas.
- Accent color: A soft sky blue (#A1BBFF), used for interactive elements, highlights, and to provide clear visual contrast.
- Headline and body text font: 'Inter' (sans-serif), for a modern, objective, and highly legible experience across all content types.
- Use a clean set of minimalist, geometric line icons for navigation, social links, and section indicators to maintain a polished, contemporary feel.
- Responsive design with a clear, sectioned layout, utilizing grid structures for project showcases on larger screens and a single-column flow for mobile accessibility.
- Incorporate subtle hover effects on actionable elements like buttons and project cards, along with smooth transitions for section scrolling to enhance user engagement without distraction.
